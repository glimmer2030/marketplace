# Weijan Marketplace

A collection of agent skills and plugins for Claude Code, Codex CLI, and other [Agent Skills](https://agentskills.io)-compatible agents.

## Installation

### Option A — Agent Skills (`npx skills`)

Works with Claude Code, Codex CLI, Gemini CLI, and other supported agents.

Install all skills:

```bash
npx skills add glimmer2030/marketplace
```

Install a specific skill:

```bash
npx skills add glimmer2030/marketplace --skill generator
```

For a specific agent:

```bash
npx skills add glimmer2030/marketplace -a codex
```

### Option B — Claude Code Plugin

Register this marketplace:

```bash
/plugin marketplace add glimmer2030/marketplace
```

Install a plugin from this marketplace:

```bash
/plugin install 5g-note-generator@glimmer2030/marketplace
```

## Available Skills

| Skill | Description | Slash Command |
|---|---|---|
| [generator](plugins/5g-note-generator/skills/generator/) | Generate 5G/O-RAN technical learning notes in Traditional Chinese | `/generator` |
